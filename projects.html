Multiplayer Prototype

A fast-paced shooter prototype built in Unity using Netcode for GameObjects. Focused on synchronization of player state, authoritative hit detection, and client-side prediction for smooth movement.

Tech stack

  • Unity 2021+, C#
  • Netcode for GameObjects, custom reconciliation
  • Basic rollback-friendly input buffering for abilities

Key challenges

  • Designing an authority model that reduces perceived latency without opening exploitation vectors.
  • Balancing network update rates vs bandwidth on mobile.

Lessons learned

Prediction + reconciliation tuned to the most frequent interactions (movement and fire) gives the best perceived responsiveness. Decoupling cosmetic animation updates from authoritative state reduced network churn.
